# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1.

In short: be respectful, assume good faith, and keep discussion constructive. Harassment or
abuse of any kind is not tolerated. Maintainers may remove, edit, or reject contributions and
comments that violate this standard.

Report concerns privately via the channel in [SECURITY.md](SECURITY.md).
